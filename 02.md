# Variable Type

## INPUT
```
#include <stdio.h>

void main(){
	int ID=534;
	char name[]="gurman";
	int Fees=5000;
	char Grade='A';
	printf("Student_ID : %d",ID);
	printf("\n Student_Name : %s",name);
	printf("\n Student_fees : %d",Fees);
	printf("\n Student_Grade : %c",Grade);
}
```

## OUTPUT
```
Student_ID : 534
 Student_Name : gurman
 Student_fees : 5000
 Student_Grade : A
```
